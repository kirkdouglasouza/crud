# crud

Essa API foi desenvolvida para criar, alterar, listar e deletar.

## Tecnologias Utilizadas

  - Node.js
  - Express
  - Sequelize (ORM)
  - MYSQL
 
  - ## Como Executar o Projeto

### Pré-requisitos

- Node.js
- npm 
- Sequelize Client

 Clone o repositório:
```sh
   git clone https://github.com/seu-usuario/lumi-projeto.git
```
### Backend
```sh
   cd crud
   npm install
   npm run dev
```
